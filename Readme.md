# Paste App Cli

CLI ile terminal çıktılarını [paste.ahmetbarut.net](https://paste.ahmetbarut.net)'e yükleyin.

## Kurulum

```bash
composer global require ahmetbarut/paste-cli
```

## Kullanım

```bash
paste-cli create "İçerik"

# veya

paste-cli create "$(cat dosya.txt)"
```
